Confidentiality Policy - REPEATX3

Last update: 2025-10-18

English version

Short summary:
This application does not collect, transmit, or share any personal data. All user data (save file only) remains only on the user’s device. This application doesn't connect to the Internet.

Details

  • Data collection: The app collects no personal or usage data.
  • Storage: The only file created is a local backup stored on the user’s device. The app does not upload these files to any server or cloud service.
  • Sharing: No data is shared with third parties.
  • Permissions: No special permissions required.
  • Security: Files are stored locally; the user is responsible for device security.
  • Changes: If this policy changes, an updated version with the date will be published here.
  • Contact: uhgoo.contact@gmail.com
